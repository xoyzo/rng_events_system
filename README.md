# RNG Events System (BallsDex Selfhost)

Adds:
- Global + local events
- Seasonal scheduling
- Spawn multipliers
- Special/regime modifiers
- Admin control system
- Discord announcements

## Install

Add to `config/extra.toml`:

[[ballsdex.packages]]
location = "/code/extra/rng_events_system"
path = "rng_events_system"
enabled = true
editable = true
